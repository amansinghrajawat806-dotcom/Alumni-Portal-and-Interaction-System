# Alumni Portal and Interaction System

A responsive **Alumni Portal and Interaction System** developed using **HTML, CSS, Bootstrap, and JavaScript**. The project is designed to connect alumni with the institution by providing a platform for registration, communication, event updates, and alumni engagement.

## Features

- Responsive Home Page
- Alumni Registration
- Login Page
- Newsroom Section
- Events Section
- Gallery
- Members/Alumni Directory
- Contact Us Page
- Mobile-Friendly Design
- Feedback Section

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
